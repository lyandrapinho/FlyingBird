# 🐦 FlyingBird

A simple Flappy Bird-style 2D game built with Python and Pygame.

---

## 🎮 Gameplay

Navigate the bird through pipes without hitting them or the screen edges.
The further you go, the higher your score!

---

## ⌨️ Controls

| Screen      | Key             | Action              |
|-------------|-----------------|---------------------|
| Menu        | `↑` / `W`       | Navigate up         |
| Menu        | `↓` / `S`       | Navigate down       |
| Menu        | `Enter`         | Confirm             |
| How to Play | `Backspace` / `Esc` | Return to menu  |
| Game        | `Space`         | Bird flies up       |

---

## ✨ Features

- Animated bird sprite with rotation effect
- Parallax scrolling background with 5 layers
- Randomly generated pipes
- Background music for menu and gameplay
- How to Play screen
- High score tracking
